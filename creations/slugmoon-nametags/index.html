Slugmoon Nametags

Slugmoon Nametags is a overhead display name system for games. It is designed to be easy to use and integrate into your existing games.

Features​

  • Easy to use and integrate.
  • Support for multiple tags on a single player.
How can I do this?
game.ReplicatedStorage.NametagAPI:Fire(Player instance, Tag text, Tag image)
warning

You cant change the colour of the tag directly, but you can fire the server then change the colour of the tag via your own script.

We will be supporting tag colour changing in the future.

  • Well documented.

And way much more!
