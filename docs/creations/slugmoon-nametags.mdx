# Slugmoon Nametags

<span className="emoji-like-icon"><img src="/img/icon.png" alt="Slugmoon Emblem" /></span> <GradientText animated>Slugmoon Nametags</GradientText> is a overhead display name system for games. It is designed to be easy to use and integrate into your existing games.

## Features

- :check: Easy to use and integrate.
- :check: Support for multiple tags on a single player.
<details>
  <summary>:info: How can I do this?</summary>
```lua
game.ReplicatedStorage.NametagAPI:Fire(Player instance, Tag text, Tag image)
```
:::warning
You cant change the colour of the tag directly, but you can fire the server then change the colour of the tag via your own script.

:info: We will be supporting tag colour changing in the future.
:::
</details>

- :book: Well documented.

:check: And way much more! :check:
